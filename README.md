This project uses a Machine Learning model to predict whether an object detected by sonar signals is a rock or a mine. The model is trained using the Sonar dataset, which contains information about sonar signals reflected from different objects underwater.
In this project, I used Python and Logistic Regression to build a classification model that can identify the object based on the sonar signal data.
What this project does
Loads and explores the sonar dataset
Splits the data into training and testing sets
Trains a Logistic Regression model
Predicts whether the object is a Rock (R) or Mine (M)
Technologies used
Python
NumPy
Pandas
Scikit-learn
Goal
The goal of this project is to practice machine learning classification and understand how models can be used to make predictions from real-world data.
